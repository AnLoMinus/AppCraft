### ⚡️AppCraft — יצירתיות חכמה באפליקציות

**שם אנגלי:** AppCraft
**שם עברי:** מלאכת־אפליקציות
**ראשי תיבות:** AC

---

## 🌌 חזון המיזם

AppCraft נולד כרעיון לחבר בין **יצירתיות, טכנולוגיה ותודעה**.
זהו מאגר ומסגרת פיתוח ליצירת אפליקציות חכמות, יפות וקלות לתחזוקה —
שכל אחת מהן תהיה כמו **יצירת אמנות טכנולוגית חיה** 🎨⚙️.

---

## 🧭 מבנה המיזם

| אזור                  | תיאור                                                         | דוגמה              |
| --------------------- | ------------------------------------------------------------- | ------------------ |
| 🧠 **Core Framework** | ליבת פיתוח אחידה הכוללת תבניות (Boilerplates), ממשקים ו־API   | `AppCraft_Core`    |
| 🎨 **UI Themes**      | ספריית עיצוב גרפית – Bootstrap, Tailwind, Neon UI, GlassMorph | `AppCraft_UI`      |
| 🧩 **Modules**        | תוספים לפי תחומים (רוחני, ניהולי, מוזיקלי, לימודי וכו׳)       | `AppCraft_Modules` |
| ⚡ **AI & Automation** | שילוב בינה, תזכורות, ניתוחים ולומדות אישיות                   | `AppCraft_AI`      |
| 🧰 **DevTools**       | כלי פיתוח, בדיקות, CI/CD, GitHub Actions, ניהול גרסאות        | `AppCraft_Tools`   |

---

## 🚀 עקרונות עיצוב

1. **מודולריות מוחלטת** – כל רכיב ניתן להחלפה מיידית.
2. **עיצוב קוסמי ניאוני** – שילוב של צבעים זוהרים, אפקטי זכוכית וברקים.
3. **התפתחות הדרגתית** – כל גרסה משתפרת בתבונה.
4. **ממשק אנושי־מלאכי** – שימוש ב-AI כשיקוף פנימי ולא כתחליף אנושי.
5. **נגישות רוחנית־טכנולוגית** – אפליקציות שנוגעות גם בנפש וגם בקוד.
